# 🏥 Smart Clinic Management System – User Stories

## Admin User Stories
1. As an admin, I want to create and manage doctor accounts so that I can control who has access to the system.
2. As an admin, I want to view system usage statistics so that I can monitor performance and user activity.
3. As an admin, I want to manage patient records so that I can ensure data accuracy and compliance.

## Doctor User Stories
1. As a doctor, I want to set my availability so that patients can book appointments during my working hours.
2. As a doctor, I want to view my upcoming appointments so that I can prepare for each consultation.
3. As a doctor, I want to access patient medical histories so that I can provide informed care.

## Patient User Stories
1. As a patient, I want to register and log in so that I can securely access my account.
2. As a patient, I want to book appointments with available doctors so that I can receive medical care.
3. As a patient, I want to view and update my personal information so that my records stay current.
